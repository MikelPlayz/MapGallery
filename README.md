# MapGallery

A custom, discord integrated, map art plugin. Originally created for the yoursecondhome minecraft server.

Their Website: [Link](https://yoursecondhome.net)





## Note

This plugin isn't meant for everyone and is purely made to meet specific requirements requested by a minecraft server I thoroughly enjoy. Additions are likely not to be added unless requested by server's admins. However I do plan to maintain this through the minecraft updates, but note not much will change from the main operation/look of this plugin.

## Documentation

### Permissions:

mapgallery.gallery
To open the gallery GUI.

mapgallery.give
Give a map by galleryID

mapgallery.admin
Remove maps from the gallery by their ID.

mapgallery.reload
For the plugin reload logic.


### Commands:

/gallery give

/gallery search

/gallery reload

/gallery remove

If you need the commands explained... Look elsewhere to be honest, this plugin isn't for you.
